Guide de démarrage rapide de Novelist

Bienvenue sur Novelist, l'application ultime pour planifier, écrire et organiser vos romans. Ce guide vous expliquera les bases pour que vous puissiez commencer votre aventure narrative en toute confiance.

1. Bibliothèque : Votre collection de livres

Lorsque vous ouvrez l'application, vous trouverez votre Bibliothèque—un hub central pour gérer tous vos projets.

  • Créer un livre : Appuyez sur le bouton “+”. Choisissez le modèle de Roman standard ou un modèle personnalisé basé sur un livre existant.
  • Modifier, définir une couverture ou supprimer un livre : Appuyez sur le menu à trois points pour accéder aux options.
  • Livres illimités : Il n’y a pas de limite au nombre de livres que vous pouvez créer !
  • Indicateurs de sécurité des livres : Les livres avec un indicateur vert sont sécurisés, tandis que ceux avec un indicateur rouge doivent être sauvegardés pour éviter de perdre votre travail.

Appuyez sur un livre pour accéder à l’espace de travail principal de Novelist.

2. Quatre sections essentielles

Le cœur de Novelist est divisé en Esquisser, Écrire, Organiser et Programmer, chacune ayant un rôle clé :

2.1 Esquisser : Planifiez les éléments de votre histoire

C’est ici que vous créez et gérez les éléments principaux de votre histoire : Thèmes, Personnages, Événements, et plus encore.

  • Ajouter des éléments : Appuyez sur le bouton “+” pour créer un nouvel élément. Chaque élément nécessite un titre (obligatoire) et éventuellement un synopsis.
  • Personnaliser les catégories : Créez, modifiez ou supprimez des catégories selon votre style d’écriture (par ex., ajoutez “Factions” ou “Planètes”).
  • Organiser les éléments : Utilisez le cercle de statut pour suivre la progression de chaque élément (par ex., Brouillon, Terminé). Appuyez longuement sur le cercle pour supprimer ou fusionner des éléments. Appuyez longuement sur un élément pour les réorganiser.
  • Déplacer des éléments : Assignez un élément à une autre catégorie ou divisez-le en deux.
  • Filtrer et rechercher : Trouvez rapidement des éléments grâce aux filtres ou mots-clés.
  • Vue détaillée d’un élément : Appuyez sur un élément pour modifier son titre, résumé, noms, tags, images, métadonnées, relations, notes et descriptions en texte enrichi.

2.2 Écrire : Rédigez vos scènes

Les scènes sont la base de votre roman, contenant le contenu réel de votre histoire.

  • Créer des scènes : Appuyez sur le bouton “+” pour ajouter une nouvelle scène avec un titre (obligatoire) et éventuellement un synopsis.
  • Éditeur de texte enrichi : Rédigez votre histoire en utilisant des options de mise en forme comme gras, italique, souligné, et plus encore. Mentionnez des éléments avec le symbole “@”. Ajoutez des liens et des commentaires.
  • Gérer les scènes : Attachez des éléments de l’Intrigue, divisez les scènes en deux, ou réorganisez-les dans la liste en appuyant longuement.
  • Suivre la progression : Utilisez le cercle de statut pour suivre le développement de chaque scène.
  • Vue détaillée d’une scène : Modifiez le titre, le résumé, les sous-intrigues, le texte, les notes, les images, et les éléments d’Intrigue associés.
👉 Important : Seul le texte rédigé dans les scènes apparaîtra dans votre manuscrit final.

2.3 Organiser : Structurez votre roman

La section Organiser vous aide à structurer votre livre en le divisant en actes, parties et chapitres.

  • Ajouter des sections : Créez une structure en arborescence pour diviser votre livre en chapitres ou autres divisions.
  • Attacher des scènes : Sélectionnez des scènes de l’écran Écriture pour les inclure dans chaque section.
  • Réorganiser les sections : Réordonnez facilement les chapitres ou sous-sections pour ajuster le flux de votre livre.
👉 Important : Seules les scènes attachées aux sections sont incluses dans le manuscrit.

2.4 Programmer : Fixez des objectifs et suivez vos progrès

Restez motivé avec des objectifs clairs et suivez votre progression au fil du temps.

  • Fixer des objectifs : Choisissez un objectif de nombre de mots, une échéance, ou les deux.
  • Progrès quotidien : Consultez un graphique de votre nombre de mots par jour pour rester responsable.

3. Aperçu et exportation de votre livre

  • Aperçu : Appuyez sur l’icône en forme d’œil pour voir à quoi ressemblera votre manuscrit final.
  • Exporter votre livre : Appuyez sur Exporter et partager dans le menu principal pour créer votre manuscrit final au format EPUB, ODT ou HTML. Partagez-le immédiatement via les applications disponibles.

4. Sauvegarde et synchronisation : Protégez votre travail

  • Sauvegardes : Enregistrez vos données localement ou sur le cloud (Google Drive ou iCloud).
  • Synchronisation manuelle : Utilisez Envoyer au cloud et Recevoir du cloud pour transférer manuellement le contenu entre appareils.
👉 Important : Effectuez des sauvegardes fréquentes pour éviter de perdre votre travail à cause de problèmes d’appareil.

5. Paramètres : Personnalisez votre expérience d’écriture

  • Mode sombre : Activez cette option pour un confort d’écriture dans des conditions de faible luminosité.
  • Explorez d’autres paramètres pour adapter l’application à vos préférences.

6. Conseils pour les nouveaux utilisateurs

  • Commencez petit : Expérimentez avec une nouvelle courte ou un plan pour vous familiariser avec les fonctionnalités.
  • Utilisez les modèles : Commencez avec le modèle de Roman standard et personnalisez-le selon vos besoins.
  • Aperçu régulier : Vérifiez vos progrès et votre structure.
  • Sauvegardez souvent : Ne prenez pas le risque de perdre votre travail.

Vous êtes maintenant prêt à commencer votre aventure d’écriture avec Novelist ! Prenez le temps d’explorer chaque fonctionnalité, et souvenez-vous : votre créativité est la seule limite. Bonne écriture ! ✍️
