Guía Rápida de Novelist

Bienvenido a Novelist, la aplicación definitiva para planificar, escribir y organizar tus novelas. Esta guía te llevará por lo básico para que puedas empezar tu viaje creativo con confianza.

1. Estantería: Tu Biblioteca de Libros

Al abrir la aplicación, encontrarás tu Estantería, un centro principal para gestionar todos tus proyectos.

  • Crear un libro: Toca el botón “+”. Elige la plantilla de Novela estándar o una plantilla personalizada basada en un libro existente.
  • Editar, establecer portada o eliminar libro: Pulsa el menú de tres puntos para ver las opciones.
  • Libros ilimitados: ¡No hay límite en la cantidad de libros que puedes crear!
  • Indicadores de seguridad de libros: Los libros con un indicador verde están seguros, mientras que los que tienen un indicador rojo deben ser respaldados para evitar la pérdida del trabajo.

Toca un libro para acceder al espacio de trabajo principal de Novelist.

2. Cuatro Secciones Esenciales

El núcleo de Novelist se divide en Planifica, Escribe, Organiza y Programa, cada una con un propósito clave:

2.1 Planifica: Planifica los Pilares de Tu Historia

Aquí creas y gestionas los elementos principales de tu historia: Temas, Personajes, Eventos y más.

  • Agregar elementos: Toca el botón “+” para crear un nuevo elemento. Cada elemento necesita un título (obligatorio) y, opcionalmente, una sinopsis.
  • Personalizar categorías: Crea, edita o elimina categorías para adaptarlas a tu estilo de escritura (por ejemplo, añade “Facciones” o “Planetas”).
  • Organizar elementos: Usa el círculo de estado para rastrear el progreso de cada elemento (por ejemplo, Primer Borrador, Completado). Mantén presionado el círculo para eliminar o fusionar elementos. Mantén presionado un elemento para reordenarlo.
  • Mover elementos: Asigna un elemento a otra categoría o divídelo en dos.
  • Filtrar y buscar: Encuentra rápidamente elementos usando filtros o palabras clave.
  • Vista detallada de elementos: Toca un elemento para editar su título, sinopsis, nombres, etiquetas, imágenes, metadatos, relaciones, notas y descripciones de texto enriquecido.

2.2 Escribe: Redacta Tus Escenas

Las escenas son la columna vertebral de tu novela, ya que contienen el contenido real de tu historia.

  • Crear escenas: Toca el botón “+” para añadir una nueva escena con un título (obligatorio) y, opcionalmente, una sinopsis.
  • Editor de texto enriquecido: Escribe tu historia utilizando opciones de formato como negrita, cursiva, subrayado y más. Menciona elementos usando el símbolo “@”. Añade enlaces y comentarios.
  • Gestionar escenas: Adjunta elementos de Trama, divide escenas en dos o reordénalas en la lista manteniendo presionado.
  • Rastrear progreso: Usa el círculo de estado para controlar el desarrollo de cada escena.
  • Vista detallada de escena: Edita el título, sinopsis, subtramas, texto, notas, imágenes y elementos relacionados de Trama.
👉 Importante: Solo el texto escrito en las escenas aparecerá en tu manuscrito final.

2.3 Organiza: Estructura Tu Novela

La sección Organiza te ayuda a dar forma a tu libro en una estructura coherente, dividiéndolo en actos, partes y capítulos.

  • Agregar secciones: Crea una estructura en forma de árbol para dividir tu libro en capítulos u otras divisiones.
  • Adjuntar escenas: Elige escenas desde la pantalla de Escribir que pertenezcan a cada sección.
  • Reorganizar secciones: Reordena fácilmente capítulos u otras subdivisiones para ajustar el flujo de tu libro.
👉 Importante: Solo las escenas adjuntadas a secciones se incluirán en el manuscrito.

2.4 Programa: Establece Metas y Sigue el Progreso

Mantente motivado con metas claras y rastrea tu progreso de escritura con el tiempo.

  • Establecer metas: Elige un objetivo de recuento de palabras, una fecha límite o ambos.
  • Progreso diario: Ve un gráfico de tu recuento de palabras cada día para mantenerte responsable.

3. Vista Previa y Exporta Tu Libro

  • Vista previa: Toca el icono de ojo para ver cómo se verá tu manuscrito al terminar.
  • Exportar tu libro: Toca Exportar y compartir en el menú principal para crear tu manuscrito final en formato EPUB, ODT o HTML. Compártelo directamente desde las aplicaciones disponibles.

4. Copia de Seguridad y Sincronización: Protege Tu Trabajo

  • Copias de seguridad: Guarda tus datos localmente o en la nube (Google Drive o iCloud).
  • Sincronización manual: Usa Enviar a la nube y Recibir de la nube para transferir contenido entre dispositivos manualmente.
👉 Importante: Haz copias de seguridad frecuentes para evitar perder tu trabajo debido a problemas con el dispositivo.

5. Configuración: Personaliza Tu Experiencia de Escritura

  • Modo oscuro: Actívalo para escribir cómodamente en condiciones de poca luz.
  • Explora otras configuraciones para ajustar la aplicación a tus preferencias.

6. Consejos para Nuevos Usuarios

  • Empieza pequeño: Experimenta con un cuento corto o un esquema para familiarizarte con las funciones.
  • Usa plantillas: Comienza con la plantilla Novela estándar y personalízala según sea necesario.
  • Previsualiza con frecuencia: Revisa tu progreso y estructura.
  • Haz copias de seguridad a menudo: No arriesgues perder tu trabajo.

¡Ahora estás listo para comenzar tu aventura de escritura con Novelist! Tómate tu tiempo para explorar cada función y recuerda: tu creatividad es el único límite. ¡Feliz escritura! ✍️
